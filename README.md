# Perfume-BrainWave-Analytics
A Tableau data modeling project mapping the neurological effects of different perfume compounds on EEG brainwaves.
### Live Dashboard
* **Tableau Public Dashboard:** https://public.tableau.com/authoring/PerfumeBrainWaveVisualisation/Sheet1#1

### Core Insights
* **Relaxation (Alpha Waves):** Specific compounds like Lavender and Chamomile systematically trigger Alpha wave increases, representing deep relaxation.
* **Alertness (Beta Waves):** Identified stimulant olfactory inputs that correlate with heightened beta-wave activity.
