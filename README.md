# Perfume-BrainWave-Analytics
[View Live Tableau Data Dashboard] (https://public.tableau.com/app/profile/aliyah.ogunyemi/viz/PerfumeBrainWaveVisualisation/Sheet1)

## Datasets Used
This project joins three primary datasets sourced from Kaggle:
1. **`perfume_Brainfunc.csv`**: Maps odorant materials to specific EEG brainwave frequency changes (e.g., Alpha, Beta, Theta).
2. **`perfume_compound.csv`**: Contains botanical sources and major chemical constituents for each fragrance material.
3. **`perfume_Psycho.csv`**: Details the psychophysiological effects (e.g., mood changes, stress reduction, alertness) linked to olfactory stimuli.

---

##  Data Modeling & Features
* **Multi-Source Data Integration:** Established relationships across datasets in **Tableau Public** using `Odorant Materials` as the key linking field.
* **Interactive Tooltips:** Configured detailed hover insights displaying chemical compounds and psychological outcomes without cluttering visual channels.
* **Heatmap / Categorical Analysis:** Colored and mapped brainwave responses to quickly isolate high-impact olfactory stimulants.
* **Agile Workflow:** Managed project tasks and dataset pipeline updates via **GitHub Projects / Kanban**.
